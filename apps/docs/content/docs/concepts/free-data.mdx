**Los datos oficiales crudos siempre son gratuitos.** Sin API key, sin cuota, sin registro. Solo pagas por la **capa de inteligencia** (búsqueda + recuperación + síntesis).

## Prueba Rápida (Sin Key)

```bash
# Todos estos funcionan sin ninguna API key:
curl https://api.intel.dom.gob/v1/institutions
curl https://api.intel.dom.gob/v1/sil/camara/iniciativas?q=compras%20p%C3%BAblicas
curl https://api.intel.dom.gob/v1/sil/senado/search?q=compras%20p%C3%BAblicas
curl https://api.intel.dom.gob/v1/graph?entity=Ley%20340-06
curl https://api.intel.dom.gob/v1/health
```

## Filosofía

> **Los datos oficiales son un bien público. La síntesis es un servicio de pago.**

Medimos la **capa de inteligencia** (búsqueda + recuperación + síntesis por LLM + validación), nunca la **capa de datos** (fuentes oficiales en bruto). Esto asegura que periodistas, investigadores, ciudadanos y la sociedad civil siempre tengan acceso gratuito a la evidencia cruda.

**Raw official data is always free.** No API key, no quota, no signup. You only pay for the **intelligence layer** (search + retrieval + synthesis).

## Always Free Endpoints

### Source Registry & Discovery

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/v1/institutions` | GET | List all registered government source plugins |
| `/v1/url-tree` | GET | Categorized source registry (SIL, Senado, portals, etc.) |
| `/v1/models` | GET | List available AI models |

## Philosophy

> **Official data is a public good. Synthesis is a paid service.**

We meter the **intelligence layer** (search + retrieval + LLM synthesis + validation), never the **data layer** (raw official sources). This ensures journalists, researchers, citizens, and civil society always have free access to the raw evidence.

INTEL.DOM.GOB utiliza un modelo de niveles simple. **Los datos oficiales crudos siempre son gratuitos.** Pagas por la **capa de inteligencia** (búsqueda + recuperación + síntesis).

## Comparación de Niveles

| Plan | Audiencia | Consultas/día | Streaming | Workflows | Docs/Intel | Precio |
|------|----------|-------------|-----------|-----------|------------|-------|
| **Público** | Cualquiera, sin registro | 20 | ❌ | ❌ | ❌ | Gratis |
| **Investigador** | Periodistas, investigadores, académicos, `.gob.do` | 200 | ✅ | ❌ | ❌ | Gratis |
| **Pro** | Empresas, firmas de abogados, analistas | 1,000+ (excedente medido) | ✅ | ✅ | ✅ | Tarifa fija/mes |
| **Institucional** | Ministerios, grandes organizaciones, integradores | Personalizado | ✅ | ✅ + SLA | ✅ | Contrato |

## Qué Siempre es Gratis (Sin Key, Sin Cuota)

Los datos oficiales crudos nunca requieren key, cuota o pago:

- Registro de fuentes: `GET /v1/institutions`, `GET /v1/url-tree`
- SIL Congreso: `GET /v1/sil/camara/*`, `GET /v1/sil/senado/*`
- Noticias y búsqueda del Senado: `GET /v1/senado/news`, `GET /v1/sil/senado/search`
- Búsqueda pública: `GET /v1/sil/senado/search`, `GET /v1/sil/senado/communities`
- Grafo de conocimiento (lectura): `GET /v1/graph`
- Sistema: `GET /v1/health`, `GET /v1/docs`, `GET /v1/openapi.json`

> **Los datos oficiales son un bien público. La síntesis es un servicio de pago.** Medimos lo segundo, nunca lo primero.

## Cómo Actualizar

| Ruta | Pasos |
|------|-------|
| **Instancia pública** | Abre Admin Console → API Keys → Crear Key → elegir plan |
| **Auto-hospedado** | Configura `REQUIRE_API_KEY=true` en `.env` → crea keys en Admin Console |
| **Verificación de Investigador** | Dominio `.gob.do` / universidad acreditada se califica automáticamente; periodistas independientes y sociedad civil aplican para aprobación manual |

## Headers de Límite de Tasa

Cada respuesta incluye:

```
X-RateLimit-Limit: 200
X-RateLimit-Remaining: 187
X-RateLimit-Reset: 1700000000
```

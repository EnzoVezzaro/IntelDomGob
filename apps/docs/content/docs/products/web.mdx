## How It Works

The Web client is an **Express server** (`apps/web/src/index.ts`) serving static HTML/CSS/JS from `public/`. It proxies API calls to the internal API gateway:

```
Browser ŌåÆ Web (Express :4200) ŌåÆ API Gateway (api:4000) ŌåÆ Orchestrator ŌåÆ Agents
```

The demo form POSTs to `/query` via the SDK (`createClient({ baseUrl })`) and renders the streaming or buffered response.

## No-JS Fallback

If JavaScript is disabled:
- Form submits via standard POST
- Server renders result via template
- No streaming, but full result works

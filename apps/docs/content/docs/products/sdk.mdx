El **SDK** es el cliente oficial de TypeScript para INTEL.DOM.GOB. Funciona de manera idéntica en Node.js y navegadores, y cada cliente oficial (Studio, Web, CLI, servidor MCP, Admin) lo utiliza.

## Instalación

```bash
npm i @intel.dom.gob/sdk
# o
pnpm add @intel.dom.gob/sdk
```

## Inicio rápido

```ts
import { createIntelClient } from "@intel.dom.gob/sdk";

const client = createIntelClient({
  baseUrl: "https://api.intel.dom.gob",   // o http://api.localhost
  token: "idg_xxx",                       // opcional en modo preview
  product: "my-app"                       // identifica tu superficie
});

// Ejecutar una consulta de inteligencia completa
const result = await client.query({
  query: "Últimas leyes sobre transparencia en RD",
  responseLang: "es"
});

console.log(result.summary);
console.log(result.citations);
```

## Métodos principales

| Método | Retorna | Descripción |
|--------|---------|-------------|
| `health()` | `Promise<HealthStatus>` | Verificación de actividad |
| `query(req)` | `Promise<IntelligenceResult>` | Consulta multi-agente completa |
| `queryStream(req, onEvent)` | `Promise<void>` | Consulta con streaming SSE |
| `chat(req)` | `Promise<{reply:string}>` | Chat de seguimiento con contexto |
| `streamChat(opts, onEvent)` | `Promise<void>` | Chat con streaming |
| `listInstitutions()` | `Promise<InstitutionDescriptor[]>` | Plugins de fuentes |
| `searchInstitution(id, q)` | `Promise<...>` | Buscar en una institución específica |
| `getUrlTree(opts)` | `Promise<{portals}>` | Registro de fuentes |
| `fetchUrl(url, opts)` | `Promise<FetchedPage>` | Extraer texto legible de una URL |
| `entities.extract(text)` | `Promise<EntitiesResult>` | Extracción de entidades nombradas |
| `documents.process(...)` | `Promise<...>` | Pipeline de inteligencia documental |
| `workflows.create(...)` | `Promise<Workflow>` | Crear workflow DAG |
| `tools.list()` | `Promise<ToolDescriptor[]>` | Catálogo de herramientas MCP |
| `prompts.get(key)` | `Promise<PromptTemplate>` | Plantilla de prompt |
| `evaluate.faithfulness(ans, ctx)` | `Promise<EvalResult>` | Evaluación de fidelidad |
| `openApi()` | `Promise<OpenAPISpec>` | Especificación OpenAPI completa |

## Opciones de consulta

```ts
await client.query({
  query: "string (requerido)",
  institutions?: string[],           // plugins específicos, omitir para automático
  model?: string,                    // sobreescribir modelo por defecto
  provider?: "gemini" | "openai" | "ollama" | "custom",
  apiKey?: string,                   // sobreescribir por consulta
  scope?: "all" | "sil" | "legislativo" | "senate" | "camara" | "senate-news" | "camara-news" | "diputado",
  search?: {                         // ajuste de búsqueda
    lang?: string;
    category?: string;
    safe?: boolean;
    timeRange?: string;
    engines?: string;
    maxResults?: number;
  },
  responseLang?: string;
});
```

## Streaming

```ts
for await (const event of client.queryStream({
  query: "Ley 340-06 procedimientos excepción"
}, (e) => console.log(e.type, e.data))) {
  // tipos de evento: plan, search, token, result, done
}
```

## Chat (con contexto)

```ts
const result = await client.query({ query: "Ley 340-06" });

await client.chat({
  systemInstruction: "Eres un asistente legal.",
  grounding: result.detailedAnalysis,
  message: "¿Qué dice sobre procedimientos de excepción?",
  history: [{ role: "user", content: "Ley 340-06" }],
  model: "gemini-3.1-flash-lite"
});
```

## Manejo de errores

```ts
import { IntelApiError } from "@intel.dom.gob/sdk";

try {
  await client.query({ query: "..." });
} catch (e) {
  if (e instanceof IntelApiError) {
    console.error(e.status, e.message, e.details);
  }
}
```

## Ejecutar el SDK localmente

El SDK es un paquete TypeScript puro — no requiere paso de compilación para los consumidores. Solo instala e importa.

```bash
# En tu proyecto
npm i @intel.dom.gob/sdk
```

Si quieres desarrollar el SDK en sí:

```bash
cd packages/sdk
npm install
npm run build   # tsc
npm run test
```

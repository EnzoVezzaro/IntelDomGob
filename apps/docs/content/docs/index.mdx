# INTEL.DOM.GOB

**Plataforma de Inteligencia Gubernamental del Estado Dominicano** — API-first, multi-agente, basada en evidencia oficial.

## ¿Qué es?

Una plataforma de inteligencia unificada que conecta agentes de IA con **fuentes oficiales del gobierno dominicano** (SIL Congreso, Senado, gacetas oficiales, datos abiertos) y entrega **respuestas estructuradas, citadas y verificables** — no alucinaciones.

```
Question → Planner → Search → Retrieval → Evidence → Synthesis → Cited Answer
```

**Cada afirmación cita su fuente.** Sin cajas negras.

## Diferenciadores Clave

| Característica | Descripción |
|----------------|-------------|
| **Solo Fuentes Oficiales** | SIL Congreso, Senado, Gaceta Oficial, portales de datos abiertos — sin scraping web random |
| **Respuestas Citadas** | Cada dato enlaza a un documento oficial (iniciativa SIL, resolución del Senado, decreto de Gaceta) |
| **Pipeline Multi-Agente** | Planner → Search → Retrieval → Evidence → Validation → Response |
| **Nativo MCP** | Servidor completo Model Context Protocol — funciona con Cursor, Claude Desktop, Continue, nuestra CLI |
| **Acceso Escalonado** | Público (gratis, sin clave) → Investigador (gratis, verificado) → Pro (con medición) → Institucional (SLA) |
| **Auto-Hospedable** | Un solo `docker compose up` — dev/prod idénticos, Caddy auto-TLS |

## Pruébalo Ahora (Sin Instalar)

```bash
# Public instance — 20 queries/day, no key
curl -X POST https://api.intel.dom.gob/v1/query \
  -H "Content-Type: application/json" \
  -d '{"query": "¿Qué leyes regulan las compras públicas en RD?"}'
```

O abre la demo en vivo: **[web.intel.dom.gob](https://web.intel.dom.gob)**

## Auto-Hospedaje en 2 Minutos

```bash
git clone <repo-url> intel.dom.gob
cd intel.dom.gob
cp .env.example .env
# Set DEFAULT_AI_API_KEY or DEFAULT_AI_PROVIDER=ollama
./scripts/up.sh
```

Abre `http://studio.localhost` — workspace multi-agente completo corriendo en local.

## Arquitectura a simple vista

```
Client (Studio, Web, CLI, MCP, SDK)
        │
        ▼
    MCP Server (Port 4100) ── JSON-RPC 2.0 over Streamable HTTP + SSE
        │
        ▼
   API Gateway (Port 4000) ── /v1/query, /v1/chat, /v1/tools, ...
        │
        ▼
Orchestrator (Planner → Search → Retrieval → Evidence → Synthesis)
        │
        ├──▶ Providers: SearXNG, Brave, Tavily, Exa
        ├──▶ AI: Gemini, OpenAI, Anthropic, DeepSeek, Ollama, Custom
        ├──▶ Institutions: 8 SIL/Senado plugins
        ├──▶ Knowledge Graph + Vector DB (ChromaDB)
        └──▶ Document Intelligence: OCR → Entities → Embeddings → KG
```

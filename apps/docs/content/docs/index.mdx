# INTEL.DOM.GOB

**Plataforma de Inteligencia Gubernamental del Estado Dominicano** — API-first, multi-agente, basada en evidencia oficial.

## What Is It?

A unified intelligence platform that connects AI agents to **official Dominican government sources** (Congress SIL, Senate, official gazettes, open data) and delivers **structured, cited, verifiable answers** — not hallucinations.

```
Question → Planner → Search → Retrieval → Evidence → Synthesis → Cited Answer
```

**Every claim cites its source.** No black boxes.

## Key Differentiators

| Feature | Description |
|---------|-------------|
| **Official Sources Only** | SIL Congress, Senate, Gaceta Oficial, open data portals — no random web scraping |
| **Cited Answers** | Every fact links to an official document (SIL initiative, Senate resolution, Gaceta decree) |
| **Multi-Agent Pipeline** | Planner → Search → Retrieval → Evidence → Validation → Response |
| **MCP Native** | Full Model Context Protocol server — works with Cursor, Claude Desktop, Continue, our CLI |
| **Tiered Access** | Público (free, no key) → Investigador (free, verified) → Pro (metered) → Institucional (SLA) |
| **Self-Hostable** | Single `docker compose up` — identical dev/prod, Caddy auto-TLS |

## Try It Now (No Install)

```bash
# Public instance — 20 queries/day, no key
curl -X POST https://api.intel.dom.gob/v1/query \
  -H "Content-Type: application/json" \
  -d '{"query": "¿Qué leyes regulan las compras públicas en RD?"}'
```

Or open the live demo: **[web.intel.dom.gob](https://web.intel.dom.gob)**

## Self-Host in 2 Minutes

```bash
git clone <repo-url> intel.dom.gob
cd intel.dom.gob
cp .env.example .env
# Set DEFAULT_AI_API_KEY or DEFAULT_AI_PROVIDER=ollama
./scripts/up.sh
```

Open `http://studio.localhost` — full multi-agent workspace running locally.

## Architecture at a Glance

```
Client (Studio, Web, CLI, MCP, SDK)
        │
        ▼
    MCP Server (Port 4100) ── JSON-RPC 2.0 over Streamable HTTP + SSE
        │
        ▼
   API Gateway (Port 4000) ── /v1/query, /v1/chat, /v1/tools, ...
        │
        ▼
Orchestrator (Planner → Search → Retrieval → Evidence → Synthesis)
        │
        ├──▶ Providers: SearXNG, Brave, Tavily, Exa
        ├──▶ AI: Gemini, OpenAI, Anthropic, DeepSeek, Ollama, Custom
        ├──▶ Institutions: 8 SIL/Senado plugins
        ├──▶ Knowledge Graph + Vector DB (ChromaDB)
        └──▶ Document Intelligence: OCR → Entities → Embeddings → KG
```

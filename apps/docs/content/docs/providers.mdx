## AI Providers

A single unified configuration selects and authenticates the default provider:
`DEFAULT_AI_PROVIDER`, `DEFAULT_AI_API_KEY`, `DEFAULT_AI_MODEL`, and `DEFAULT_BASE_URL`.

| Provider | `DEFAULT_AI_PROVIDER` | Notes |
|----------|-----------------------|-------|
| **Gemini** (default) | `gemini` | Google Gemini (with `stream()`) |
| **OpenAI** | `openai` | OpenAI GPT models (`DEFAULT_BASE_URL` optional) |
| **Anthropic** | `anthropic` | Anthropic Claude models |
| **DeepSeek** | `deepseek` | DeepSeek models (`DEFAULT_BASE_URL` optional) |
| **Ollama** | `ollama` | Local models (`DEFAULT_BASE_URL`) |
| **Custom** | `custom` | Any OpenAI-compatible endpoint via `DEFAULT_BASE_URL` |

## How to Add a Provider

### Search Provider

1. Create `providers/brave/` with `package.json` depending on `@intel.dom.gob/providers`
2. Implement `SearchProvider.search()`
3. Register in `apps/api/src/index.ts`
4. Add `"brave"` to `SEARCH_PROVIDERS` in `.env`

### AI Provider

1. Create `providers/openai/` with `package.json`
2. Implement `AiProvider.generate()` (and `stream()` optionally)
3. Register in `apps/api/src/index.ts`
4. Set `DEFAULT_AI_PROVIDER` (+ `DEFAULT_AI_API_KEY` / `DEFAULT_BASE_URL`) in `.env`

### Provider Interface

```ts
export class XProvider implements SearchProvider | AiProvider | OcrProvider | PresentationProvider {
  id: string;
  kind: "search" | "ai" | "ocr" | "presentation";
  label: string;
  enabled = true;
  
  // implement search() / generate() / extractText() / render()
}
```

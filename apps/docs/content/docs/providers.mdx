Providers are the **only** code that touches external systems. Adding a new provider requires only creating a new implementation — nothing else changes.

---

## Search Providers

| Provider | Default | Env Var | Description |
|----------|---------|---------|-------------|
| **SearXNG** | ✅ | — | Self-hosted metasearch engine (always on) |
| **Brave** | optional | `BRAVE_API_KEY` | Brave Search API |
| **Tavily** | optional | `TAVILY_API_KEY` | Tavily Search API |
| **Exa** | optional | `EXA_API_KEY` | Exa neural search |

## How to Add a Provider

### Search Provider

1. Create `providers/brave/` with `package.json` depending on `@intel.dom.gob/providers`
2. Implement `SearchProvider.search()`
3. Register in `apps/api/src/index.ts`
4. Add `"brave"` to `SEARCH_PROVIDERS` in `.env`

### AI Provider

1. Create `providers/openai/` with `package.json`
2. Implement `AiProvider.generate()` (and `stream()` optionally)
3. Register in `apps/api/src/index.ts`
4. Add `"openai"` to `AI_PROVIDERS` / set `DEFAULT_AI_PROVIDER`

### Provider Interface

```ts
export class XProvider implements SearchProvider | AiProvider | OcrProvider | PresentationProvider {
  id: string;
  kind: "search" | "ai" | "ocr" | "presentation";
  label: string;
  enabled = true;
  
  // implement search() / generate() / extractText() / render()
}
```

## Productos de INTEL.DOM.GOB

INTEL.DOM.GOB no es una sola aplicación — es un ecosistema de productos para inteligencia gubernamental de la República Dominicana.

| Producto | Qué es | Para quién | Estado |
|----------|--------|------------|--------|
| **API** | Gateway API-first — cada capacidad expuesta vía REST + SSE | Desarrolladores, integradores | En vivo |
| **Studio** | Workspace de IA multi-agente (fork AGPL-3.0 de Odysseus) | Usuarios avanzados, investigadores | En vivo |
| **MCP** | Servidor Model Context Protocol — herramientas vía Streamable HTTP + SSE | Cualquier cliente compatible con MCP | En vivo |
| **Web** | Cliente web ligero sin JavaScript para consultas rápidas | Cualquiera, sin instalación | En vivo |
| **CLI** | Cliente terminal interactivo (MCP) | Desarrolladores, automatización | En vivo |
| **Admin** | Consola de operador para claves, cuotas, tenants | Operadores | En vivo |
| **Docs** | Este sitio de documentación | Todos | En vivo |

---

## API

El punto de entrada para todo. Versionada (`/v1`), REST + streaming SSE, compatible con OpenAI. Cada cliente anterior se comunica con ella a través del SDK.

- **Core**: `query`, `query/stream`, `chat`, `institutions`, `url-tree`
- **Compatible con OpenAI**: `chat/completions`, `models`, `embeddings`
- **Legislativo (SIL)**: Endpoints de Cámara de Diputados + Senado
- **Servicios de inteligencia**: inteligencia documental, entidades, knowledge graph
- **Motor de workflows**: ejecución DAG con intervención humana

Consulta la [Referencia de la API](/docs/api) para los +40 endpoints.

## Studio

Un workspace de IA auto-hospedado — chat, agentes, investigación profunda, documentos, correo, notas, calendario y workflows con modelos locales. Se conecta a INTEL.DOM.GOB **solo vía el servidor MCP**.

- Fork de [Odysseus](https://github.com/pewdiepie-archdaemon/odysseus) (AGPL-3.0)
- Multi-usuario con autenticación bcrypt, cookies de sesión, TOTP 2FA, RBAC
- Servidores MCP integrados para correo, memoria, RAG, generación de imágenes, navegador
- Ejecuta en FastAPI + Uvicorn + SQLite + ChromaDB

## MCP

Un servidor Model Context Protocol que expone +20 herramientas: consultas de inteligencia, datos SIL, Senado DSpace, datos institucionales. Cualquier cliente MCP estándar (Claude Desktop, Studio, CLI, agentes personalizados) se conecta de la misma manera.

- **Streamable HTTP**: `http://mcp.localhost/mcp`
- **SSE heredado**: `http://mcp.localhost/sse`
- **JSON-RPC heredado**: `POST http://mcp.localhost/`

## Web

Un cliente web renderizado por servidor, sin JavaScript. Lista instituciones gubernamentales y ejecuta consultas de inteligencia. Sin paso de compilación, sin JS del lado del cliente — una implementación de referencia del SDK.

- Ejecuta en el puerto `4200` (Docker: `web.localhost`)
- Usa solo `listInstitutions()` y `query()`

## CLI

Un cliente terminal interactivo modelado en OpenCode. Se conecta al servidor MCP y expone todas las capacidades de la plataforma mediante un bucle de prompt. Sin lógica de API, sin SDK — solo MCP.

```bash
intel                 # terminal interactivo
intel -p "pregunta"   # one-shot
```

## Admin

Consola de operador para gestionar API keys, cuotas, tenants y RBAC/ABAC. Inicia sesión con la clave de administrador (`INTEL_API_TOKEN`) creada en el primer arranque.
